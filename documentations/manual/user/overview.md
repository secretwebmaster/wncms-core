# User Dashboard Overview

This section is for non-technical website operators.

Use it to manage daily content and settings from the backend UI.


## Start Here

1. Open backend sidebar
2. Go to the module you need
3. Follow the matching guide below


## Content Guides

- `Posts List`: see `./dashboard/posts.md`
- `Links List`: see `./dashboard/links.md`
- `Pages List`: see `./dashboard/pages.md`
- `Users List`: see `./dashboard/users.md`

## System Guide

- `System Settings`: see `./dashboard/settings.md`


## Recommended Daily Flow

1. Check `Posts List` and `Pages List` for content updates
2. Check `Links List` for broken/outdated links
3. Check `Users List` for new users or role changes
4. Open `System Settings` only when you need site-wide changes
