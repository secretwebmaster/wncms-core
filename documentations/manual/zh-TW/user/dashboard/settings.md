# 設定介面說明

## System 分頁版面

- 在桌面檢視（`lg` 以上）下，**System** 分頁的基礎欄位會以每列 2 個項目顯示。
- 表單間距已收斂（`p-3 p-lg-4` + 較小列距），提高資訊密度。

## 多站點網站選擇器

- 當模型網站模式為 **multi** 時，網站核取方塊改為全寬顯示。
- 每個網域各佔一列，便於瀏覽與勾選。

## Multisite 分頁版面

- 在設定 `multisite` 分頁中，模型網站模式改為在右側面板（`col-lg-8`）以每列 2 項網格顯示，不再使用表格。
- 每個項目採用模型名稱與模式下拉同一行顯示（`global` / `single` / `multi`）。

## 行動版分頁導覽

- 設定分頁導覽在行動版改為單列顯示。
- 分頁不再換行，支援橫向捲動。
- 捲動條已隱藏，介面更精簡。
- 頁面載入與切換分頁時，會自動捲動至目前啟用分頁位置。
