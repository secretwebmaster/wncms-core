# 設定

本指南用於協助非技術人員在後台調整系統設定。


## 設定入口

後台選單：

1. 進入 **系統設定**
2. 使用左側分頁切換設定群組

常見分頁：

- `基礎設定`
- `登入設定`
- `SMTP設定`
- `Cloudflare 設定`
- `緩存設定`
- `社交設定`
- `頁面設定`
- `采集接口設定`
- `內容模組設定`
- `用戶設定`
- `管理員設定`
- `統計設定`
- `翻譯設定`
- `多站點設定`（啟用多站點時）
- `API設定`
- `開發者設定`（啟用開發者模式時）


## 日常常用操作

## 1）修改網站基礎資料

1. 打開 **系統設定**
2. 點擊 `基礎設定`
3. 修改站點名稱/描述/關鍵字等欄位
4. 點擊 **儲存全部**

預期結果：

- 全站基礎資料更新

## 2）開啟或關閉一個開關項

1. 進入對應分頁（例如 `用戶設定`）
2. 開啟/關閉目標開關
3. 點擊 **儲存全部**

預期結果：

- 對應功能立即依設定生效

## 3）設定 SMTP 郵件發送

1. 進入 `SMTP設定`
2. 填寫 SMTP 資料（主機、埠號、帳號、密碼、寄件者）
3. 點擊 **儲存全部**

預期結果：

- 郵件相關功能使用新的 SMTP 設定

## 4）設定多站點模型模式（如有）

1. 進入 `多站點設定`
2. 為各模型選擇模式：
- `全域`
- `單站`
- `多站`
3. 點擊 **儲存全部**

預期結果：

- 模型站點範圍依所選模式更新

## 5）檢查核心更新

1. 在左側底部點擊 **檢查更新**
2. 依更新頁面提示操作

預期結果：

- 可查看目前可用的核心更新


## 常見問題

## 修改後未生效

- 確認已點擊 **儲存全部**
- 重新整理頁面確認數值仍存在
- 如果環境有快取，請執行快取清理

## 找不到某個分頁

- 部分分頁僅在相關功能啟用後顯示
- `開發者設定` 僅在開發者模式開啟時顯示
