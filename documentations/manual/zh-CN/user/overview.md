# 用户后台总览

本区面向非技术网站维护人员。

用于在后台完成日常内容与系统管理。


## 从这里开始

1. 打开后台侧边栏
2. 进入对应功能模块
3. 按下面对应指南操作


## 内容管理指南

- `文章列表`：见 `./dashboard/posts.md`
- `连结列表`：见 `./dashboard/links.md`
- `页面列表`：见 `./dashboard/pages.md`
- `用户列表`：见 `./dashboard/users.md`

## 系统管理指南

- `系统设定`：见 `./dashboard/settings.md`


## 建议的每日流程

1. 先检查 `文章列表` 与 `页面列表`
2. 再检查 `连结列表` 是否有失效连结
3. 在 `用户列表` 处理新用户与角色
4. 仅在需要全站调整时进入 `系统设定`
