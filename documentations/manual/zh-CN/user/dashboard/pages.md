# 页面

本指南用于帮助非技术人员在后台管理页面。


## 页面管理入口

后台菜单：

1. 进入 **页面列表**
2. 需要时使用顶部筛选（`搜索`、`显示详情`、`显示封面`）


## 日常常用操作

## 1）新增页面

1. 打开 **页面列表**
2. 点击 **新增页面**
3. 在 `基础` 分页填写：
- `标题`
- `代称（显示于 URL）`
- `类型`（`plain`、`template` 或其他可选类型）
4. 若类型是 `template`，选择 `可用页面模板`
5. 在右侧 `发布相关` 设置：
- `状态`
- `可见性`
6. 点击主提交按钮（`发布`）

预期结果：

- 页面出现在 **页面列表**
- 保存后可使用预览按钮

## 2）编辑页面内容

1. 在 **页面列表** 点击 **编辑**
2. 修改 `标题`、`内容`、`备注` 或模板选项
3. 点击提交按钮（`更新`）

预期结果：

- 页面修改成功并可预览最新内容

## 3）设置页面属性

1. 打开页面编辑页
2. 在右侧找到 **页面属性**
3. 切换选项（例如 `hide_title`）
4. 点击提交按钮

预期结果：

- 前台页面显示方式按设置变化

## 4）批量删除页面

1. 在 **页面列表** 勾选目标页面
2. 点击 **批量删除**
3. 确认弹窗

预期结果：

- 选中页面被删除


## 常见问题

## 页面地址打不开

- 检查 `代称` 是否为空或冲突
- 检查 `状态` 与 `可见性`
- 先用 `预览` 按钮确认后台保存正常

## 看不到模板选项分页

- 只有 `类型` 为 `template` 时才会显示 `模板选项`
- 切换类型后先保存一次，再重新进入编辑页
