# 连结

本指南用于帮助非技术人员在后台管理连结。


## 连结管理入口

后台菜单：

1. 进入 **连结列表**
2. 需要时使用顶部筛选（`搜索`、分类筛选、`显示详情`）


## 日常常用操作

## 1）新增连结

1. 打开 **连结列表**
2. 点击 **新增连结**
3. 填写必填字段：
- `状态`
- `名称`
- `URL`
4. 建议补充：
- `连结分类`
- `连结标籤`
- `备注`
- `连结图标` / `连结封面`
5. 点击 **新增**

预期结果：

- 新连结出现在 **连结列表**

## 2）编辑连结

1. 在 **连结列表** 点击 **编辑**
2. 修改字段（例如名称、URL、标籤、图标、置顶/推荐）
3. 点击 **编辑** 保存

预期结果：

- 连结资料立即更新

## 3）批量调整状态或推荐属性

1. 在 **连结列表** 勾选多条
2. 使用以下按钮：
- `设为启用`
- `设为停用`
- `设置置顶`
- `取消置顶`
- `设为推荐`
- `取消推荐`
3. 在弹窗确认

预期结果：

- 多条连结一次完成更新

## 4）批量处理连结标籤

1. 勾选多条连结
2. 点击 **处理链接标签**
3. 在弹窗设置：
- `操作`（`attach`、`detach`、`sync`）
- `分类`
- `标籤`
4. 点击 **提交**

预期结果：

- 选中连结的分类/标籤完成更新

## 5）批量编辑字段

1. 勾选目标连结
2. 点击 **批量编辑**
3. 填写要统一修改的字段
4. 提交

预期结果：

- 选中连结按填写字段批量更新


## 常见问题

## 前台看不到连结

- 检查 `状态` 是否为可显示状态
- 检查 `结束日期` 是否已过期
- 检查前台分类/标籤筛选条件

## 图标或封面未更新

- 重新上传图片后再保存一次
- 确认图片格式为支持类型（`png`、`jpg`、`jpeg`、`gif`）
