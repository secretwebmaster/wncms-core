# 用户

本指南用于帮助非技术人员在后台管理会员帐号。


## 用户管理入口

后台菜单：

1. 进入 **用户列表**
2. 使用筛选（`搜索`、`角色`、`显示详情`、`显示积分`）快速定位用户


## 日常常用操作

## 1）新增用户

1. 打开 **用户列表**
2. 点击 **新增用户**
3. 填写必填字段：
- `角色`
- `用户名`
- `Email`
- `密码`
- `确认密码`
4. 点击 **新增**

预期结果：

- 新用户出现在 **用户列表**

## 2）编辑用户角色或基础资料

1. 在 **用户列表** 点击 **编辑**
2. 修改字段，例如：
- `角色`
- `用户名`
- `Email`
- `密码`（仅在需要重置时填写）
3. 点击 **编辑** 保存

预期结果：

- 用户资料立即更新

## 3）查看更多用户资料

1. 打开筛选中的 `显示详情`
2. 查看扩展栏位（API 密钥、语言地区、时区、登入资讯）

预期结果：

- 列表显示完整帐号资讯

## 4）快速查看用户积分

1. 打开 `显示积分`
2. 查看积分栏位
3. 需要充值时，点击积分后方的 `+`

预期结果：

- 可在列表快速查看与处理积分

## 5）删除用户

1. 在目标用户行点击 **删除**
2. 在弹窗确认

预期结果：

- 用户从列表移除


## 常见问题

## 无法新增用户

- 确认必填项都已填写
- 确认 `用户名` 与 `Email` 未重复

## 编辑后用户无法登入

- 检查用户角色是否正确
- 如果修改了密码，请把新密码提供给用户
