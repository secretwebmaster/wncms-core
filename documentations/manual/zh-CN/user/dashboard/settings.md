# 设置界面说明

## System 页签布局

- 在桌面视图（`lg` 以上）下，**System** 页签的基础字段会以每行 2 个项目显示。
- 表单间距已收紧（`p-3 p-lg-4` + 更小行距），提升信息密度。

## 多站点网站选择器

- 当模型网站模式为 **multi** 时，网站复选框改为全宽显示。
- 每个域名单独占一行，便于浏览与勾选。

## Multisite 页签布局

- 在设置 `multisite` 页签中，模型网站模式改为在右侧面板（`col-lg-8`）以每行 2 项网格显示，不再使用表格。
- 每个项目采用模型名称与模式下拉同一行显示（`global` / `single` / `multi`）。

## 移动端页签导航

- 设置页签导航在移动端改为单行显示。
- 页签不再换行，支持横向滚动。
- 滚动条已隐藏，界面更简洁。
- 页面加载与切换页签时，会自动滚动到当前激活页签位置。

## Quick Link 行为

- 页面工具栏星标按钮会先按当前 URL 判断是否已收藏，若无再回退到路由名判断。
- Quick Link 存储前会进行规范化与去重。
- Superadmin 角色名称为 `superadmin`（无连字符），工具栏显示判断已与此一致。
