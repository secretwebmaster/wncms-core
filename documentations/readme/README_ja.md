# Wncms コアパッケージ

このリポジトリは **WNCMS のコアパッケージ (wncms-core)** であり、WNCMS 全体を支える基盤クラス、マネージャー、ヘルパー、サービスプロバイダ、バックエンド構造、マイグレーションなどを提供します。

本パッケージは **単体で動作する CMS ではありません**。  
WNCMS プラットフォームを構築するための下層フレームワークです。

WNCMS 全体の紹介、インストール方法、テーマシステム、プラグインシステムなどの詳細は、メインプロジェクトの README.md を参照してください。

## 特徴

- 統一された ModelManager アーキテクチャ
- 翻訳・タグ・キャッシュ・設定などのコアサービス
- バックエンドコントローラとビュー、開発用スキャフォールド
- テーマの読み込み・検証・設定サポート
- プラグインロード機能
- API ヘルパーと Resource Transformer
- インストール処理と環境初期化ツール
- モデルやマネージャーを自由に上書き可能な拡張性
