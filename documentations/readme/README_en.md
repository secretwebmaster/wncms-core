# Wncms Core Package

This repository contains the **core package of WNCMS**, providing all fundamental classes, managers, helpers, service providers, migrations, and backend structures required by the WNCMS ecosystem.

It is **not** a standalone CMS.  
It is the underlying framework that powers the full WNCMS platform.

For the complete WNCMS system introduction, installation guide, theme system, plugin system, and full documentation, please refer to the main project repository.

## Features

- Unified ModelManager architecture
- Translation, tagging, caching, and option handling core services
- Backend controllers, views, and scaffolding
- Theme loading and validation utilities
- Plugin loading support
- API helpers and resource transformers
- Installation utilities and environment setup handlers
- Extensible architecture that allows applications to override any model or manager
